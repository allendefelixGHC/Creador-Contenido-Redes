<task type="checkpoint:human-action">
  <name>Task 1: Regenerate Azure SAS with delete permission</name>
  <action>
  The current Azure SAS token has `sp=cw` (create + write) but Phase 9 needs `sp=cwrd` (create + write + read + delete) for blob cleanup after publish.

  Run this command in Azure CLI (or Azure Cloud Shell):

  ```bash
  # Get storage account key
  ACCOUNT_KEY=$(az storage account keys list --account-name propulsarcontent --resource-group propulsar --query '[0].value' -o tsv)

  # Generate new SAS with delete permission
  az storage account generate-sas \
    --account-name propulsarcontent \
    --account-key "$ACCOUNT_KEY" \
    --services b \
    --resource-types co \
    --permissions cwrdl \
    --expiry 2027-04-10T00:00:00Z \
    --https-only \
    -o tsv
  ```

  Then update the `AZURE_SAS_PARAMS` environment variable in the n8n Azure Container App:
  - Azure Portal → Container Apps → propulsar-n8n → Settings → Environment variables
  - Update `AZURE_SAS_PARAMS` with the new SAS query string (everything after the `?`)
  - Save and wait for container to restart

  Verify by running a test DELETE against a known test blob:
  ```bash
  curl -I -X DELETE "https://propulsarcontent.blob.core.windows.net/posts/test-delete.txt?<NEW_SAS>" \
    -H "x-ms-version: 2020-10-02" \
    -H "x-ms-date: $(date -u '+%a, %d %b %Y %H:%M:%S GMT')"
  ```
  Expected: 404 (blob doesn't exist) or 202 (deleted). NOT 403.
