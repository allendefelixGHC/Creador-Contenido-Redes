Locate the existing node with `"id": "prep-rehost-input"` (around line 435 of `n8n/workflow.json`). Modify its `parameters.jsCode` field to add a carousel rejection guard near the top. The new guard:

  throw new Error('Carousel publishing not yet supported — Phase 7 pending. Received format: ' + format);

This means the existing `if (format === 'carousel' && Array.isArray(data.image_urls)) { ... }` branch will now be DEAD CODE (unreachable) but leave it in place — Phase 7 will remove the guard and the branch becomes reachable again. The current jsCode remains logically correct for single-post because only the else-if (`data.final_image_url`) path will execute.

Because the jsCode lives in a single-string JSON field (escaped newlines), the cleanest way is to rewrite the entire `jsCode` value. Final value should be (as a JSON string with `\n` escapes):

```javascript
// Normalize both single-post and carousel briefs into the sub-workflow's expected input shape.
// Single post: $json has final_image_url (string).
// Carousel: $json has image_urls (array of strings) and num_images.
//
// Output shape for Execute Workflow:
//   { image_urls: [{ index, url }], post_id, approval_number }

const data = $input.first().json;
const format = data.format || 'single';

// Phase 5 guard: single-post only. Phase 7 removes this when carousel publish lands.
if (format === 'carousel') {
  throw new Error('Carousel publishing not yet supported — Phase 7 pending. Received format: ' + format);
}

let imageUrls = [];
if (format === 'carousel' && Array.isArray(data.image_urls)) {
  imageUrls = data.image_urls.map((url, i) => ({ index: i + 1, url }));
} else if (data.final_image_url) {
  imageUrls = [{ index: 1, url: data.final_image_url }];
} else {
  throw new Error('Prep Re-host Input: no image URL found on approved post (neither final_image_url nor image_urls)');
}

// post_id: derive from session/timestamp. Prefer an existing session_id if present (set by Recuperar sesion Supabase).
const postId = data.session_id || ('propulsar_' + Date.now());

return [{
  json: {
    // Pass ALL upstream fields through so downstream nodes (Sheets Log, future Meta publish)
    // still have access to topic, type, angle, instagram/facebook text, etc.
    ...data,
    // Plus the sub-workflow input fields:
    image_urls: imageUrls,
    post_id: postId,
    approval_number: data.approval_number
  }
}];
```

Also update the `notes` field of this node to mention the guard: `"Normalizes single-post (final_image_url) into the sub-workflow input shape. Carousel briefs are rejected with a runtime error (Phase 7 removes this guard). Passes all other fields through so downstream nodes retain topic/type/angle/text."`
