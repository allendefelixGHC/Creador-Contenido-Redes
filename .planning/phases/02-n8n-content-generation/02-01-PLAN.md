**Node 3: Code node — "🔧 Parsear prompts carrusel"**
Same type as existing "🔧 Parsear contenido" (`n8n-nodes-base.code`, typeVersion 2).

JavaScript code (Run Once for All Items mode):
```javascript
const raw = $input.first().json.message.content;
const b = $('🎯 Webhook Trigger').first().json.body;

let parsed;
try {
  parsed = JSON.parse(raw.replace(/```json\n?/g,'').replace(/```\n?/g,'').trim());
} catch(e) {
  throw new Error('JSON inválido de GPT-4o (carrusel): ' + raw.substring(0,200));
}

const slides = parsed.slides || [];

if (slides.length !== b.num_images) {
  console.warn(`⚠ GPT-4o devolvió ${slides.length} slides, se esperaban ${b.num_images}`);
}

return [{
  json: {
    instagram:       { caption: parsed.instagram_caption || '' },
    facebook:        { caption: parsed.facebook_caption || '' },
    topic:           b.topic,
    type:            b.type,
    angle:           b.angle || null,
    platforms:       b.platforms,
    image_model:     'ideogram',
    fal_model_id:    null,
    has_own_image:   false,
    image_url:       null,
    has_text_in_image: true,
    approval_number: b.approval_number,
    timestamp:       b.timestamp,
    format:          'carousel',
    num_images:      b.num_images,
    estructura:      parsed.estructura || 'listicle',
    image_prompts:   slides.map(s => ({
      slide_num:     s.slide_num,
      texto_overlay: s.texto_overlay,
      prompt:        s.prompt,
    })),
  }
}];
```

Position: [860, 520].

IMPORTANT: The Code node uses `$('🎯 Webhook Trigger')` to access original brief data — the SAME node-reference-by-name pattern proven in the existing "🔧 Parsear contenido" node. It also includes `instagram` and `facebook` caption objects in the output to mirror the single-post "Parsear contenido" output shape, enabling Phase 3 to use captions without special-casing.

Note on count validation: Use `console.warn` (not `throw`) if slide count doesn't match. The user will see the preview via WhatsApp and can reject — hard failure is too brittle for a preview-and-approve flow.
