Leave the rest of the Code node unchanged — it already handles both single-post (final_image_url) and carousel (image_urls array) formats correctly.

**B. Insert IF node `🔀 ¿Formato Carrusel?` after `🔗 Merge Rehost Output`:**

Create a new IF node (typeVersion 1, NOT v2 — v2 is broken in n8n 2.14.2):
- Name: `🔀 ¿Formato Carrusel?`
- typeVersion: 1
- Condition: `{{$json.format}}` equals `carousel` (string comparison, operation: equal)
- Position: Place it between `🔗 Merge Rehost Output` and the existing IG chain. X position ~3200 (just right of Merge Rehost Output which is around 3080-3100), Y ~400.

- `🔗 Merge Rehost Output` output 0 -> `🔀 ¿Formato Carrusel?` (remove existing connection from Merge Rehost Output to `📤 IG: Create Container`)

- `🔀 ¿Formato Carrusel?` output 1 (FALSE = single-post) -> `📤 IG: Create Container` (restores the existing single-post path)

NOTE on IF v1 outputs: In IF typeVersion 1, output index 0 = TRUE branch, output index 1 = FALSE branch. This is confirmed by the existing `🔀 ¿Carrusel?` IF node in the workflow which uses the same pattern.

**C. Add `💾 Guardar sesión Supabase (Carousel)` HTTP Request node on the carousel path:**

This node inserts a carousel session into Supabase BEFORE the WA preview is sent. It mirrors the existing `💾 Guardar sesión Supabase` node (which serves the single-post path).

Create an HTTP Request node:
- Name: `💾 Guardar sesión Supabase (Carousel)`
- typeVersion: 4.2
- method: POST
- url: `={{ $env.SUPABASE_URL }}/rest/v1/content_sessions`
- authentication: none (use headers)
- sendHeaders: true
- headerParameters:
  - apikey: `={{ $env.SUPABASE_ANON_KEY }}`
  - Authorization: `=Bearer {{ $env.SUPABASE_ANON_KEY }}`
  - Content-Type: `application/json`
  - Prefer: `return=representation`
- sendBody: true
- specifyBody: json
- jsonBody: `={{ JSON.stringify({ session_id: 'propulsar_' + Date.now(), approval_number: $json.approval_number, topic: $json.topic, type: $json.type, angle: $json.angle || null, platforms: $json.platforms || [], image_model: $json.image_model || 'ideogram', format: 'carousel', image_urls: $json.image_urls, instagram_caption: $json.instagram_caption || ($json.instagram && $json.instagram.caption) || null, facebook_caption: $json.facebook_caption || ($json.facebook && $json.facebook.caption) || null, status: 'pending' }) }}`
- Position: Between `🗂️ Collect Image URLs` and `📨 Split URLs WA`. X ~1500, Y ~700 (adjust to fit between existing nodes on the carousel path).
- retryOnFail: true (Supabase INSERT is idempotent by session_id, safe to retry)
